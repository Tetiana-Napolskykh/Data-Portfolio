# Project: Revenue Metrics

This project demonstrates revenue analysis for a project and the creation of a dashboard for product managers. The dashboard allows tracking the dynamics of Revenue and Paid Users and analyzing the factors affecting their changes.

## Objective
- Track Monthly Recurring Revenue (MRR)
- Analyze new and churned users
- Calculate metrics such as ARPPU, Churn Rate, Expansion/Contraction MRR, and others
- Visualize data using Tableau

## Structure
- `task_description.md` – description of the task
- `solution.sql` – SQL query to retrieve the data
- `tableau_dashboard.md` – link to Tableau Public dashboard
